Echopype

Nov 1, 2020

Echopype is an open-source Python package aimed at enhancing the interoperability and scalability in processing ocean sonar data for biological information.

I started building this package in early 2018 when I couldn’t find an affordable tool that allow easy access and manipulation of echosounder data collected by different sonar models.

For the latest updates, check out our repo at: https://github.com/OSOceanAcoustics/echopype.
